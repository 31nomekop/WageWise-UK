# WageWise UK — PWA Test Build (GitHub Pages)

This is a **no-laptop** way to test WageWise UK on your iPhone using **GitHub Pages**.

## What you do
1. Create a new GitHub repository (public).
2. Upload all files in this folder to the repo root.
3. Enable GitHub Pages (Deploy from branch: main / root).
4. Open the Pages URL on your iPhone.
5. Tap Share → Add to Home Screen.

## Files
- index.html
- styles.css
- app.js
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

## Notes
- Works offline after first load (service worker).
- This is an estimator for 2025/26 PAYE.
